Sobre el proyecto

To Do List es un sistema que permite gestionar todas las tareas que tienes pendientes desde un solo sistema, existen versiones de escritorio (para ordenadores y tablets) y versiones verticales (para teñefonos y tablets). Esta puede ejecutarse como una aplicación de Android o como una página web

Información sobre el desarrollo

¿Cómo he hecho las aplicaciones?

Para programar las aplicaciones, primero las diseñé en Canva como me gustaría que fueran y después las programé en Visual Studio Code.

¿Qué lenguajes de programación he usado?

Para programar las aplicaciones usé HTML 5, CSS 5 y JavaScript.

¿Cuánto tiempo ha llevado desarrollar las aplicaciones?

Este proyecto epezó en julio del año 2024 y terminó en octubre del mismo año.

¿Cómo he publicado las versiones web?

Para la publicación de las versiones de navegador he usado la plataforma Vercel.

¿Cómo he publicado las APPs?

Para la publicación de las APPs he usado la plataforma Web To App, que permite mediante una version web o un archivo .html crear una aplicación para Android.
